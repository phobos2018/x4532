A) App Structure - App Pages - top

This app has a side navigation layout.

The app consists of pages
  1. Markets - Crypto Markets Listing Page - ./app/markets
  2. Market - Market exchange Page with buy and sell Tabs - ./app/market
  3. Balances - That page lists the coins you own on Exchange Wallet - ./balances
  4. Home - Home screen to customize - ./app/home
  5. Orders - The Order History listing - ./app/orders
  6. Settings - Linked from Side Nav Bar Blank Template Page - ./app/settings

B) Page tructure - top

Every Page has a folder with specific file types. The HTML template is in *.component.html file. The CSS is generated automatically from SCSS files which are in ./app/ folder. Styles for pages are in app.scss file.
*.routing.module.ts - contains routing for navigation
*.component.ts - contains model and data logic for the page
*.module.ts - contains the module dependencies for the pages

C) Running The App - top

- To run tns run ios nativescript requires Xcode on osX

  1. - We recommend Visual Studio Code with NativeScript Plugin to enable debbuging
  2. - app have dependency on Cocoapods to build, more info on https://docs.nativescript.org/plugins/cocoapods
  3. - General documentation for NativeScript - https://docs.nativescript.org/.

  1. *Note** you should have **node v6.5.0 or higher** and **npm 3.10.3 or higher**.
  2. * To run the NativeScript app (currently supports 3.x):
  3. npm install -g nativescript
  4. npm install -g webpack
  5. npm install
  6. tns run ios - to run ios tns run android - to run android

After you finalize those steps you can start run the app!
